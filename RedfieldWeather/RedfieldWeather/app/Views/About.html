About
Data on this site is powered by a Davis Vantage Pro/2 weather station and Weather Underground.

I am currently using Virtual Weather Station software to upload my data to Weather Undergound and many other weather sites.If you want more detail about my weather station please read my blog post http://www.kd8fwv.com/category/weather/

I have converted this site to use AngularJS and a Web API 2.0 backend for the storage and retrieval of data. I am using Microsoft Azure Storage Tables for data storage and history.

This is more than a weather site this is my test bed to try out new web technologies. I plan on making all this code avaliable at a later date. I need to complete a few more things before I can do that.
